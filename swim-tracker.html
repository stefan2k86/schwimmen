📊 Google Sheets Setup

Damit die App funktioniert, brauche ich drei Dinge von dir:

Aus der URL: docs.google.com/spreadsheets/d/HIER/edit

Erstelle einen API-Key in der Google Cloud Console (Sheets API aktivieren!)

📋 Tabellen-Struktur

Erstelle in deiner Google Tabelle zwei Tabellenblätter mit exakt diesen Namen:

Blatt 1: Tage
Spalten: ID | Datum | Bezeichnung

Blatt 2: Wettkämpfe
Spalten: ID | TagID | Disziplin | Meldezeit | Bestzeit | GeschwommeneZeit

Zeitformat: M,SS,Z → z.B. 1,23,4 = 1 Min, 23 Sek, 4 Zehntel

🔒 Tabelle öffentlich lesbar machen

Damit die App lesen kann, ohne Login: Tabelle → Freigeben → Jeder mit dem Link (Betrachter)

Zum Schreiben (Speichern) wird ein Service Account empfohlen – oder du trägst Wettkämpfe direkt in die Tabelle ein.
